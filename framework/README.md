# Framework de Producción de Proyectos

Mini sitio estático para publicar en GitHub Pages.

## Archivos

- `index.html`: página de entrada con links a las estaciones.
- `e01-captura-validacion-informacion-base.html`: Estación 01.
- `e02-definicion-producto.html`: Estación 02.

## Publicación rápida en GitHub Pages

1. Crear un repositorio en GitHub, por ejemplo: `framework-produccion-proyectos`.
2. Subir estos archivos a la raíz del repositorio.
3. Ir a **Settings > Pages**.
4. En **Build and deployment**, elegir **Deploy from a branch**.
5. Seleccionar branch `main` y carpeta `/root`.
6. Guardar.

El sitio quedará disponible con una URL similar a:

`https://TU_USUARIO.github.io/framework-produccion-proyectos/`
